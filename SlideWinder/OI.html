Visual OI Guide

Driver Left Joystick

Y-Axis: Drive (speed)
X-Axis: Drive (strafe)
Button 1: LowerElevator (while-held)
Button 2: GradualDrive (while-held)
Button 3: Shift (while-held)

Driver Right Joystick

X-Axis: Drive (angle)
Button 1: RaiseElevator (while-held)
Button 2: QuickTurn (while-held)
Button 4: OpenClaw (when-pressed)
Button 5: CloseClaw (when-pressed)

Manipulator Joystick

Left Y-Axis: ManualControl (elevator)
Right Y-Axis: ManualControl (arm raise)
Right X-Axis: ManualControl (arm deploy)
Button 1: SetElevatorPosition 1 (when-pressed), during ManualControl: ArmStab (toggle-when-pressed)
Button 2: SetElevatorPosition 2 (when-pressed), during ManualControl: ArmDeploy (toggle-when-pressed)
Button 3: SetElevatorPosition 3 (when-pressed)
Button 4: SetElevatorPosition 4 (when-pressed)
Button 5: ManualControl (while-held)
Button 6: CloseClaw (when-pressed)
Button 7: CalibrateElevator (when-pressed)
Button 8: OpenClaw (when-pressed)
Button 9: ManualControl: RaiseArm (when-pressed)
Button 10: ManualControl: LowerArm (when-pressed)
D-Pad Up: SetElevatorPosition 10.5 inches
D-Pad Right: QuickStackOne
D-Pad Down: FeedFromStation
D-Pad Left: QuickStackTwo

Autonomous Modes

Mode 0: Immobilize
Mode 1: AutoDrive
Mode 2: PullBinBack
Mode 3: StackTotes
Mode 4: AutoSlideToLine
Mode 5: AutoFollowLine
Mode 6: AutoVisionFollowLine
